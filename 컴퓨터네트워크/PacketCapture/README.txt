<<make 사용법>>
make 	   : 컴파일 (파일들을 합쳐준다.)

make clean : 오브젝트 파일과 실행파일 삭제 + 혹시모를 core파일도



<<프로그램 설명>>
프로그램 실행파일 이름: pk  (PacKet)
프로그램 실행환경: 우분투

중요: 메뉴에서 항목 선택하면 자식 프로세스가 생성되고 패킷 캡처를 시작한다.
이때 패킷 캡처를 그만두고싶다면 키보드에서 아무문자를 누르고 엔터를 치면된다.
그러면 다시 메뉴로 돌아가서 항목을 다시 선택할 수 있다.
-> 윈도우의 system("pause");를 사용하지 못해 인터넷에서 방법을 찾아서
    아무키나 누르면 되게 하려 했지만, 실패했다(코드는 주석으로 해서 남아있다)

각 프로토콜 별 printf()를 담당하는 부분: dns.c, http.c, ftp.c, telnet.c
위 네개의 파일에서 쓰이는 공통적인 부분: common_func.c -> IP, TCP, UDP, Payload 담당
모든 .c파일이 포함하는 헤더파일: superheader.h -> 모든 전처리가 모여있다

superhead.h에 dns.h, http.h 등 모든 사용자 정의 헤더파일도 포함시킴
-> 각 .c에서는 superhead.h만 포함시키면 됨

main.c에서는 위 파일들의 함수들을 호출하며 실질적인 프로그램 동작 담당
